# Library Management System

A Java-based Library Management System developed using JDBC and MySQL.

## Features

- Add Book
- View Books
- Search Book
- Add Member
- Issue Book
- Return Book
- View Issued Books
- Delete Book

## Technologies Used

- Java
- JDBC
- MySQL
- OOP
- DAO Pattern

## Database

Run `library_db.sql` before starting the application.
